# 🔍 Leitor de Configurações de Equipamentos

Script em Python para leitura e análise de arquivos binários:

- INSTALL.CXF
- INSTALL.SDW
- COEFVOL.SDW
- ALTCABOS.SDW

## 🚀 Funcionalidades

- Varredura automática de diretórios
- Leitura de arquivos binários
- Geração de logs estruturados
- Validação de integridade dos arquivos

## 🛠 Tecnologias
- Python
- Tkinter (interface para seleção de pasta)
- Manipulação de arquivos binários

## ▶️ Como usar

```bash
python main.py
